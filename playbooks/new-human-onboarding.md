# New Human Onboarding

## Pre-Start

- [ ] Provision access to GitHub repositories.
- [ ] Provision access to communication tools.
- [ ] Ensure NDA and basic agreements are executed (documents stored elsewhere).

## Day 1

- [ ] Read the mission, vision, and values.
- [ ] Read operating principles.
- [ ] Read relevant policies (security, data & privacy, AI safety, communications, code of conduct).

## Week 1

- [ ] Schedule asynchronous Q&A or meeting with the orchestrator.
- [ ] Assign initial tasks with a clear mentor or point of contact.
- [ ] Confirm access to required dashboards, runbooks, and playbooks.
